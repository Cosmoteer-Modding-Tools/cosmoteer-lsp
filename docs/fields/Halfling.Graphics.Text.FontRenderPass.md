## SizeRelativeOffset
`Vector2` · optional

The position offset of this pass relative to the pixel size of the font.

## Effects
`IFontEffect[]` · optional

A modifiable list of effects to apply to the font during this render pass. The effects will be applied in the order in which they appear in this list. Adding or removing effects only has an effect on characters that have not yet been pre-rendered.

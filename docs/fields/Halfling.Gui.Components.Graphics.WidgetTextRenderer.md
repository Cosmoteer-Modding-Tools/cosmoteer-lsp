## Text
`string` · optional

The text to be rendered. If the XmlFormatting property is true, then the text will be parsed for XML format tags.

## TextProvider
`ITextProvider` · optional

An ITextProvider object that, if not null, will keep this WidgetTextRenderer's Text property updated to match the ITextProvider.

## XmlFormatting
`bool` · optional

Whether the Text property will be parsed for XML format tags.

## Font
`Font` · optional

The font used to render the text.

## Color
`Color` · optional

The color of the text.

## BackgroundColor
`Color` · optional

The color of the background behind the text.

## Shader
`asset (shader)` · optional

The shader used to render the text, or null to use the default shader.

## ShaderConstants
`ShaderConstantCollection` · optional

A collection of constant values for the Shader. This collection may be null if there is no Shader or there are no constants set.

## FontSize
`int` · optional

The pixel size of the text font.

## FontStyle
`enum FontStyle` · optional · one of: `Regular`, `Bold`, `Italic`, `Underline`, `Strikeout`

The font style of the text.

## HAlignment
`enum HAlignment` · optional · one of: `Left`, `Center`, `Right`

How the text is aligned horizontally within its Rect.

## VAlignment
`enum VAlignment` · optional · one of: `Top`, `Center`, `Bottom`

How the text is aligned vertically within its Rect.

## InternalLineAlignment
`enum VAlignment` · optional · one of: `Top`, `Center`, `Bottom`

How text of different sizes within the same line are aligned within that line.

## OversizeMode
`enum OversizeMode` · optional · one of: `None`, `Wrap`, `Shrink`, `Ellipses`

Whether and how text that is too big to fit in its rect will be modified to fit.

## EllipsesText
`string` · optional

The text to append when OversizeMode is set to Ellipses and the main text exceeds the maximum width.

## MaxRenderedChars
`int` · optional

The maximum number of characters to render. The text will still be positioned and formatted as if all characters were rendered.

## CharacterMask
`CharacterMask` · optional

The optional CharacterMask used to change one character into another during rendering. This can be useful for implementing password fields. If null, then the character will be rendered as-is.

## Padding
`Borders` · optional

The empty space padding around the text.

## Clip
`bool` · optional

Whether the text is clipped to the rect provided by RectProvider.

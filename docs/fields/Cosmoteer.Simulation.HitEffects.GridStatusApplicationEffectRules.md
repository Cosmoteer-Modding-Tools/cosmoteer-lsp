## HitPartSourceRectType
`enum PartRectType` · optional · one of: `Normal`, `Physical`

The type of rect from the hit part to use as the set of source cells. If null, only the hit location cell is used.

## MaxCost
`number` · required

The maximum path cost to a cell for it to be affected.

## HitDirectionCostFactor
`number` · optional

The factor applied to cell cost based on the path direction's alignment with the initial hit direction. For example, a factor of 0.25 with a hit direction of (1,0) would result in 25% of the cost when pathing from cell (5,0) to (6,0), but 100% of the cost pathing from cell (5,0) to (5,1).

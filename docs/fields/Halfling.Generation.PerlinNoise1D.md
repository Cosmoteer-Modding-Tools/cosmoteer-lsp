## Seed
`int` · optional

A value used to hash the input coordinates. A PerlinNoise1D with the same seed value (and other properties) will always produce the same output for a given input coordinate.

## GridSize
`float` · optional

The size in units of the underlying Perlin noise grid.

## Repeat
`int` · optional

If not zero, the outputted values will repeat every this number of coordinate steps.

## Octaves
`int` · optional

The number of iterations used to generate the final noise value.

## Persistence
`float` · optional

The factor by which the strength each further octave iteration will be multiplied.

## OutputExponent
`float` · optional

The power to which output values are raised before they are scaled, shifted, and clamped.

## OutputScale
`float` · optional

The factor by which output values are multiplied after they are exponentiated and before they are shifted and clamped.

## OutputShift
`float` · optional

The amount added to the output values after they are exponentiated and scaled and before they are clamped.

## ClampOutput
`range<float>` · optional

The range to which output values will be clamped after they are exponentiated, scaled, and shifted.

## Holdable
`bool` · optional

Whether the user can press and hold the key to generate repeated selection changes.

## HoldInitialRepeatInterval
`Time` · optional

The amount of time between the initial press of a key and the first repeated selection change. Has no effect if Holdable is false.

## HoldRepeatInterval
`Time` · optional

The amount of time between each selection change while the user is holding a key. Has no effect if Holdable is false.

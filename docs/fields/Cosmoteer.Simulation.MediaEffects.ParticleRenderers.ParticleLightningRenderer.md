## EnableLightningShaderAnimation
`bool` · optional

Enables animating the vertex color along the entire bolt for use by the lightning shader.

## BlendLightningShaderAnimation
`bool` · optional

If lightning shader animation is enabled, enables blending between it and the original vertex color over the lifetime of the bolt.

## UseSameSeedForEntireBolt
`bool` · optional

When not enabled (the default behavior), changes the noise seed per bolt segment so each segments motion is more random.

## IgnoreBoltEndingOverride
`bool` · optional

If lightning shader animation is enabled, disables the default behavior of overriding the animation for the bolts ending segment with values to make the bolts ending fade smoothly.

## IgnoreBoltStartOverride
`bool` · optional

If lightning shader animation is enabled, disables the default behavior of overriding the animation for the bolts starting segment with values to make the bolts start fade smoothly.

## LightningShaderAnimationSegments
`float` · optional · default `1`

If lightning shader animation is enabled, controls the number of animated slices (bright sections) the bolt should have.

## TimeShiftPerSegment
`float` · optional

Shifts the perlin noise sample forward or backwards in time incrementally for each bolt segment.

## AnimationIntensity
`float` · optional · default `1`

If lightning shader animation and blending are enabled, controls the intensity of the lightning shader animation.

## SegmentOverlap
`float` · optional

Shifts the endpoint of each segment forward by this amount.

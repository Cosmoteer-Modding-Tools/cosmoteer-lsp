## VertexColor
`Color` · optional

The vertex colors of both the inner and outer edges of this circle. If getting this property and the inner and outer vertex colors are now equal, an exception will be thrown. Modifying this property will force the underlying circle quads to be recomputed.

## InnerVertexColor
`Color` · optional

Vertex color of the inner edge of the circle. Modifying this property will force the underlying circle quads to be recomputed.

## UVRect
`Rect` · optional

The UV coordinates on Texture from which the circle is rendered. The outer edge will equal UVRect.Top and the inner edge will equal UVRect.Bottom. The circle will loop from UVRect.Left to UVRect.Right. Modifying this property will force the underlying circle quads to be recomputed.

## InnerRadius
`float` · optional

The distance from the center of the circle to inside edge.

## Thickness
`float` · optional

The thickness of the circle as measured from its inner radius to its outer radius.

## QuadCount
`int` · optional

The number of quads that will be used to form the circle. The higher this value, the smoother the circle will appear. Modifying this property will force the underlying circle quads to be recomputed.

## Arc
`Arc` · optional

The angular arc of the rendered circle. Anything less than 360-degrees will result in a pie-shape. Modifying this property will force the underlying circle quads to be recomputed.

## Z
`float` · optional

The Z depth value of the rendered circle. Modifying this property will force the underlying circle quads to be recomputed.

## DashRepeat
`int` · optional

The number of times that the dashes will repeat. The default is 0.

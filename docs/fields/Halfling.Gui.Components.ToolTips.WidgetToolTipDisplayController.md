## ClearTextWhenSourceCleared
`bool` · optional

Whether the text on the widget will be set to an empty string when the tool tip source is cleared.

## ToggleWidgetActive
`bool` · optional

Whether the attached widget's SelfActive property will be toggled such that the widget is only active when there is a tool tip source set.

## AutoPosition
`bool` · optional

Whether the location and (if MatchWidth or MatchHeight are set) size of the attached widget will be automatically set by this WidgetToolTipDisplayController.

## AutoRaise
`bool` · optional

Whether the attached widget will be automatically raised to the top of its parent when shown.

## SourceCorner
`enum Corner` · optional · one of: `TopLeft`, `TopRight`, `BottomLeft`, `BottomRight`

When AutoPosition is true, gets or sets the corner of the tool tip source to which the attached widget's ToolTipCorner will be aligned.

## ToolTipCorner
`enum Corner` · optional · one of: `TopLeft`, `TopRight`, `BottomLeft`, `BottomRight`

When AutoPosition is true, gets or sets the corner of the attached widget that will be aligned with tool tip source's SourceCorner.

## PositionOffset
`Vector2` · optional

When AutoPosition is true, gets or sets the offset to apply to the position of the attached widget once it is aligned with the tool tip source.

## PreferredHorizontalSpace
`float` · optional

The amount of horizontal space that is preferred when positioning the attached widget. If the amount of space available is less than this value, then the widget may be located on the opposite side of the tool tip source if there is more space there.

## PreferredVerticalSpace
`float` · optional

The amount of vertical space that is preferred when positioning the attached widget. If the amount of space available is less than this value, then the widget may be located on the opposite side of the tool tip source if there is more space there.

## MatchWidth
`bool` · optional

When AutoPosition is true, gets or sets whether the width of the attached widget will be modified to match the width of the tool tip source.

## MatchHeight
`bool` · optional

When AutoPosition is true, gets or sets whether the height of the attached widget will be modified to match the height of the tool tip source.

## Delay
`Time` · optional

The amount of time that showing a tool tip will be delayed. This property will implicitly get or set the DelayProvider property.

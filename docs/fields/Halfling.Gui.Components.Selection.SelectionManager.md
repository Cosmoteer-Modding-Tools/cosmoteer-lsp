## MinSelected
`int` · optional

The minimum number of Widgets that may be selected at a time. Calling AttemptToDeselect() will have no effect if the number of selected Widgets is not greater than this value. Modifying this property will have no immediate effect on the currently-selected Widgets.

## MaxSelected
`int` · optional

The maximum number of Widgets that may be selected at a time. Calling AttemptToSelect() may fail or cause another Widget to become deselected depending on the value of the MaxSelectedMode property. Modifying this property will have no immediate effect on the currently-selected Widgets.

## MaxSelectedMode
`enum MaxSelectedMode` · optional · one of: `DontSelect`, `DeselectPrevious`

The action to take when AttemptToSelect() is called but the maximum number of Widgets are already selected. Modifying this property will have no immediate effect on the currently-selected Widgets.

## SelectionChangeOrder
`enum SelectionChangeOrder` · optional · one of: `DeselectThenSelect`, `SelectThenDeselect`

The order in which widgets will be selected and deselected when calling AttemptToSelect().

## File
`asset (font)` · optional

The font file to load, typically a `.ttf`, resolved relative to this file.

## Files
`asset (font)[]` · optional

Several font files loaded together, with later files supplying the glyphs earlier ones lack. Used to chain a fallback font for CJK glyphs.

## Passes
`FontRenderPass[]` · optional

The render passes applied when drawing text, in order. Each pass draws the whole text once with its own offset and [[Halfling.Graphics.Text.FontRenderPass.Effects]], which is how shadow and glow fonts layer a blurred colored copy under the foreground.

## ForceSameWidths
`[string, string]` · optional

Two equal-length strings read as character pairs: each character in the first string is forced to render with the width of the matching character in the second. Vanilla maps `"123456789"` to `"000000000"` so digits align in columns.

## AntialiasQuality
`int` · optional

The antialiasing quality level used when rasterizing glyphs.

## ResourceStorage
`→ PartComponentRules` · required

The backing resource storage that the store will use to provide and receive resources.

## ReservedResourceLevel
`float` · optional

The amount of resources in the ResourceStorage that are unavailable to the store. Represents a percentage of max capacity if ResourceLevelsAreFractions is true.

## MaxResourceLevel
`float` · optional

The resource level in the ResourceStorage that the store is able to deposit up to. Represents a percentage of max capacity if ResourceLevelsAreFractions is true.

## ResourceLevelsAreFractions
`bool` · optional

Whether the resource level values are a fraction of the ResourceStorage's capacity, or an absolute value.

## AllowProviding
`bool` · optional · default `true`

Whether the store can fulfill requests for its resource from the part-network.

## AllowReceiving
`bool` · optional · default `true`

Whether the store can receive resources pushed through the part-network.

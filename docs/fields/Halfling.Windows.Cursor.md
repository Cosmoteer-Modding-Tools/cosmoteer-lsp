## File
`asset (image)` · optional

The cursor bitmap, resolved relative to this file. Either this or [[Halfling.Windows.Cursor.OSCursor]] must be present.

## HotSpot
`IntVector2` · optional

The pixel inside the bitmap that sits on the exact mouse position, as `[x, y]` from the top-left. Defaults to the top-left corner.

## Scale
`number` · optional

A factor the cursor bitmap is scaled by when displayed.

## OSCursor
`enum OSCursor` · optional

A cursor supplied by the operating system instead of a bitmap, named by its standard role.

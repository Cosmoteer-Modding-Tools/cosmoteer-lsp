## Holdable
`bool` · optional

Whether the user can click and hold on the widget to generate repeated Clicked events.

## HoldInitialRepeatInterval
`Time` · optional

The amount of time between the initial press of the widget and the first repeated Clicked event. Has no effect if Holdable is false.

## HoldRepeatInterval
`Time` · optional

The amount of time between each Clicked event when the user is holding down the widget. Has no effect if Holdable is false.

## DoubleClickMaxInterval
`Time` · optional

The maximum amount of time between clicks that will result in a DoubleClicked event being fired.

## ClickMode
`enum ClickMode` · optional · one of: `OnRelease`, `OnPress`

Whether the player needs to press or release the input button to "click" the widget.

## HotkeyMode
`enum ClickMode` · optional · one of: `OnRelease`, `OnPress`

Whether the player needs to press or release the hotkey to "click" the widget.
